# Release Notes

This is the release candidate for the fist Canopy BMC release.

The release focuses on supporting the HPE ProLiant Gen11 platforms and is primarily intended for early adopters and evaluation processes.

## Features
- Host boot-up ([#45](https://github.com/canopybmc/canopybmc/pull/45))
- IPMI KCS ([#89](https://github.com/canopybmc/canopybmc/pull/89))
- KVM support ([#86](https://github.com/canopybmc/canopybmc/pull/86))
- Serial console and Serial-over-LAN (SOL) support ([#40](https://github.com/canopybmc/canopybmc/pull/40))
- System event (SEL) logging and dump endpoints ([#97](https://github.com/canopybmc/canopybmc/pull/97))
- BIOS version and inventory information ([#66](https://github.com/canopybmc/canopybmc/pull/66), [#74](https://github.com/canopybmc/canopybmc/pull/74))
- Power regulators via CPLD ([#46](https://github.com/canopybmc/canopybmc/pull/46))
- Fan and sensor control ([#75](https://github.com/canopybmc/canopybmc/pull/46, https://github.com/canopybmc/canopybmc/pull/75))
- Thermal zones ([#90](https://github.com/canopybmc/canopybmc/pull/90))
- PSU information and presence ([#49](https://github.com/canopybmc/canopybmc/pull/49), [#64](https://github.com/canopybmc/canopybmc/pull/64))
- CI integration and testing of essential features ([#42](https://github.com/canopybmc/canopybmc/pull/42), [#82](https://github.com/canopybmc/canopybmc/pull/82), [#104](https://github.com/canopybmc/canopybmc/pull/104))

## Core Components
- U-Boot v2026.01
- Linux Kernel 6.18.7
- OpenBMC 2026-01-11 (ad8d54b))

## Known Issues
- Undocumented mapping from bootblock to HPE ProLiant Gen11 model ([#95](https://github.com/canopybmc/canopybmc/issues/95))
- EFI BIOS reports CHIF response errors ([#92](https://github.com/canopybmc/canopybmc/issues/92))

## Not supported
- BIOS configuration ([#88](https://github.com/canopybmc/canopybmc/issues/88))
- BIOS updates ([#84](https://github.com/canopybmc/canopybmc/issues/84))
- Storage inventory ([#83](https://github.com/canopybmc/canopybmc/issues/83))
- NMI support ([#56](https://github.com/canopybmc/canopybmc/issues/56))
- IPMI over LAN
- Virtual media

> [!NOTE]
> This is a source only release.
